# Deloitte Data Analytics Project

## 📊 Overview
This project is part of the Deloitte Data Analytics Job Simulation on Forage. It focuses on analyzing machine telemetry data and identifying key business insights.

## 🎯 Objectives
- Analyze factory downtime data
- Identify most failure-prone machines
- Evaluate gender equality using Excel

## 🛠 Tools Used
- Tableau
- Microsoft Excel

## 📁 Files Included
- Tableau Dashboard (.twbx)
- Excel Dataset (.xlsx)
- Certificate (.pdf)

## 📌 Key Insights
- Factory with highest downtime: Daikibo Factory Seiko
- Most failing machine: LaserWelder
- Equality scores classified into Fair, Unfair, and Highly Discriminative
